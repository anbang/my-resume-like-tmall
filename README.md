###兼容到IE56789，全兼容，优雅降低方案

预览的地址：[http://broszhu.com/works/my-resume-like-tmall/index.html](http://broszhu.com/works/my-resume-like-tmall/index.html)


测试环境

- chrome
- IETeser中的678910
- win系统自带的IE57891011；
